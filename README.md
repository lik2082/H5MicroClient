# H5微端

H5微端的原理很简单，就是使用Android App的`WebView`控件。那么什么是WebView呢？WebView是手机中内置的一款高性能webkit内核浏览器，在SDK 中封装的一个组件。 没有提供地址栏和导航栏，WebView只是单纯的展示一个网页界面。简单地可以理解为简略版的浏览器。
